## 커밋 컨벤션

| 태그 이름 | 설명 |
| :--- | :--- |
| **Feat** | 새로운 기능을 추가할 경우 |
| **Fix** | 버그를 고친 경우 |
| **Style** | 코드 포맷 변경, 세미콜론 누락 등 코드 로직 수정이 없는 경우 |
| **Refactor** | 프로덕션 코드 리팩토링 |
| **Docs** | 문서를 수정한 경우 (README 등) |
| **Chore** | 빌드 테스트 업데이트, 패키지 매니저 설정 등 (프로덕션 코드 변경 X) |

---
**커밋 메시지 예시:** `Feat: 로그인 기능 구현`  
`Fix: 메인 페이지 이미지 로딩 오류 수정`
